# 前端开发计划

## 技术栈

- Next.js 16、React 19、TypeScript
- Tailwind CSS 4
- shadcn/ui 组件规范（MVP 当前以本地轻量组件实现，后续可逐项迁移）
- Privy、wagmi/viem、Solana Wallet Adapter（集成阶段）

## 模块

1. 设计系统：米白/墨绿主题、暗色模式、排版、卡片、状态色。
2. 首页：大师头像选择、热门议题、排行榜、钱包入口。
3. 对话：历史侧栏、实时消息、语音入口、大师动态加入/移除。
4. 委员会：阶段状态、暂停/继续、讨论轮次、投票与综合方案。
5. 投资卡片：参数编辑、风险摘要、模拟、交易预览和签名入口。
6. 个人中心：跨链资产、收益、仓位、策略、风险、授权、活动。
7. 基础设施：国际化、主题、游客本地记录、API Client、错误与加载状态。

## 里程碑

- M1：高保真静态交互与响应式页面。
- M2：后端 AI 流式接口、演示数据和状态机接入。
- M3：Privy 登录、钱包资产读取、Uniswap 报价与签名。
- M4：Aave/PancakeSwap/Jupiter 接口、监控和黑客松演示加固。

## 验收

- 可从首页完成单大师或三人大师委员会路径。
- 委员会消息实时出现且可暂停。
- 可生成并打开交易预览。
- 主题、语言、钱包弹窗与个人中心可用。
- 375px、768px、1440px 三档无横向溢出。

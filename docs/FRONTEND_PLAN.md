# 前端实现说明

更新日期：2026-06-12

## 技术栈

- Next.js 16、React 19、TypeScript
- Tailwind CSS 4
- Lucide React
- LocalStorage 本地状态

## 路由

| 路由 | 用途 |
| --- | --- |
| `/` | 人物星图、对话、历史、个人中心 |
| `/community` | 人物排行榜、搜索、上传和下载 |
| `/market` | 多链市场和研究工具 |
| `/docs` | 官方产品文档 |
| `/rankings` | 兼容旧链接，跳转 `/community` |

## 首页

- 桌面主导航：市场、投资大师、社区。
- 头像菜单：个人中心、历史对话、产品文档、开发者 API、语言、日夜模式。
- 社区不重复出现在头像菜单。
- 人物以动态圆球展示，大小按使用次数对数映射。
- 初始人物顺序和搜索结果按“官方使用数 + 当前浏览器会话使用数”降序排列。
- 支持人物搜索、单击选择、鼠标/触摸拖拽和惯性物理运动。
- 点赞和评论只在社区卡片显示。
- 选中人物与偏好刷新后保持。
- 首页展示最近 6 个历史会话。

## 对话

- 单人物或 3-8 人委员会。
- 用户可选择平台默认 AI，或启用浏览器端加密保存的 OpenAI 兼容 API 配置。
- 左侧历史栏桌面显示，移动端使用返回按钮。
- 支持暂停/继续发送。
- 显示角色票型、置信度、综合结论、共识率和最终方案。
- 支持查看投票机制和模拟执行预览。
- AI 服务失败时展示演示回复。

## 社区

- 顶部排行榜。
- 人物市场搜索。
- 本地 JSON 上传。
- Blob JSON 下载。
- 上传校验字段和风险枚举。
- 当前本地上传人物不会自动加入首页。

## 市场

- 加载、错误、刷新状态。
- ETH、BNB、SOL 卡片。
- 展示价格、TVL、更新时间；缺失数据用“—”。
- 跳转委员会并预填问题。

## 个人中心

- 模拟盘与真实钱包双模式。
- 模拟充值、提现、0.01% 手续费、流水。
- EVM/Solana 公开余额读取。
- 真实充值地址复制。
- 收费金库按钮保持禁用说明。

## 响应式和无障碍

- 支持桌面、平板、移动端。
- 交互按钮提供可访问名称。
- 支持 `prefers-reduced-motion`。
- 日夜主题使用 CSS 变量。
- 弹窗提供遮罩关闭和明确关闭按钮。

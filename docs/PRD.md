# 追光者 / Oracle Capital 产品需求文档

## 1. 产品定位

追光者是面向加密资产用户的 AI 多链投资决策与执行平台。用户可选择一位 AI 模拟投资大师对话，或选择至少三位大师组成投资委员会，通过独立分析、交叉质询、投票和置信度加权形成投资方案。

黑客松版本免费，默认模拟交易，少量白名单协议提供真实小额交易入口。

## 2. 首发范围

- 网络：Ethereum、BNB Smart Chain、Solana，支持主网/测试网切换。
- 标的：现货、DeFi、合约模拟。
- 真实协议：Uniswap、Aave、PancakeSwap、Jupiter；合约仅模拟。
- 钱包：Privy 统一接入 MetaMask、WalletConnect、Phantom、邮箱/社交嵌入式钱包。
- 大师：巴菲特、芒格、彼得·林奇、牛顿、哈耶克、马克思、亚当·斯密、凯恩斯。
- 模式：建议、受控自动、授权范围内全自动。MVP 中自动化与自有合约使用演示数据。

## 3. 核心演示路径

1. 游客进入首页，单选或多选大师。
2. 选择至少三位大师召开委员会。
3. 委员会实时展示独立分析、质询、反驳和投票；用户可暂停或修改议题。
4. 系统按置信度加权生成可编辑投资卡片。
5. 用户查看模拟结果或交易预览。
6. 连接钱包，完成风险确认并演示 Ethereum Uniswap ETH → USDT 小额交换。
7. 在个人中心查看跨链资产、收益、策略、风险与活动。

## 4. 关键规则

- 委员会最少三人，最多讨论三轮，平票时追加一轮。
- AI 根据问题自动控制回复深度，可展开深度分析。
- 用户可修改金额、仓位、杠杆、止盈止损并排除代币或协议。
- 协议须在安全白名单内，路径比较价格、滑点、Gas、收益与风险。
- 交易前展示协议、路径、Gas、滑点、授权和风险；模拟由用户选择。
- 真实交易统一展示风险摘要并要求勾选确认。
- 合规：IP 地区、用户声明、钱包风险筛查；MVP 使用可替换演示接口。
- 历史内容设计为客户端加密后存 IPFS/Filecoin，链上保存哈希与权限；MVP 本地模拟。

## 5. 非功能要求

- 完整桌面/移动响应式布局。
- 中英文自动适配并可手动切换。
- 浅色/深色主题手动切换。
- 支持减少动态效果。
- 模拟、测试网、主网状态必须清晰区分。
- 现实人物均标注为 AI 风格模拟，无本人授权或背书。

## 6. MVP 不包含

- 自有策略金库和授权智能合约。
- 真实自动化 Keeper、Gelato 或 Chainlink Automation。
- 完整大师广场审核、发布、举报流程。
- 合约真实交易。
- 生产级 IPFS/Filecoin 加密同步。

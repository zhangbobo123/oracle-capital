# 追光者 / Oracle Capital 产品需求文档

更新日期：2026-06-12

## 1. 产品定位

追光者是面向加密资产用户的 AI 多链投资研究、委员会决策和模拟执行平台。用户可选择一位 AI 思想人物单聊，或选择至少三位人物组成委员会，通过独立分析、投票和综合方案辅助决策。

所有现实人物均为思想框架模拟，不代表人物本人，不存在授权、关联或背书。网站不承诺收益。

## 2. 当前上线范围

- 网络：Ethereum、BNB Chain、Solana。
- 标的语境：现货、DeFi、合约研究；合约与交易执行仍为模拟。
- AI：DeepSeek `deepseek-chat`。
- 官方人物：巴菲特、芒格、林奇、牛顿、哈耶克、马克思、亚当·斯密、凯恩斯。
- 钱包：MetaMask/EVM 与 Phantom/Solana；WalletConnect 仅保留入口。
- 市场：ETH、BNB、SOL 价格及 Ethereum、BSC、Solana TVL。
- 资产：EVM 原生 ETH/BNB 和 Solana 原生 SOL。
- 资金：本地模拟盘；真实资产保持非托管。

## 3. 首页人物系统

### 3.1 动态人物球

- 人物以圆形球展示，保留头像、中文名、投资流派和使用次数。
- 使用次数由官方演示基数加当前浏览器历史会话次数组成。
- 球体大小公式：

```text
value = log10(uses + 1)
ratio = (value - minValue) / (maxValue - minValue)
diameter = 152 + ratio * 96
```

- 最终桌面直径约为 152-248px；移动端限制在视口宽度约 46% 以内。
- 球体具有轻微上下浮动动画；系统开启“减少动态效果”时关闭动画。

### 3.2 搜索与选择

- 搜索字段：中文名、英文名、投资流派、人物语录。
- 1 位人物可开始单人对话。
- 2 位人物时开始按钮禁用。
- 3-8 位人物可召开投资委员会。
- 最近人物选择保存到浏览器，下次进入自动恢复。

### 3.3 点赞与评论

- 点赞、评论按钮与人物选择按钮分离，互动不会误触选择。
- 点赞是当前浏览器本地状态。
- 评论最长 160 字，每人物保留最近 30 条。
- 当前没有公共点赞计数、审核、举报或服务端评论系统。

## 4. AI 对话与委员会

### 4.1 请求规则

- 最多向 API 发送 8 位人物。
- 最近上下文最多发送 8 条非系统消息。
- DeepSeek 请求超时 45 秒。
- 模型输出必须是严格 JSON。
- 不得编造实时行情、承诺收益或声称已经执行交易。

### 4.2 投票与共识

- 票型：`approve`、`abstain`、`reject`。
- 每位人物返回 0-100 的置信度。
- 共识率：

```text
round((approve + abstain * 0.5) / voters * 100)
```

- 彩蛋不参与投票，不改变共识率和最终方案。
- 未选择人物不允许临时加入委员会。

### 4.3 最终方案

最终方案包括：

- 动态标题和核心逻辑。
- 最多 6 项配置，后端归一化为 100%。
- 风险等级：稳健、均衡、进取、高风险。
- 收益判断与压力回撤。
- 委员会主要分歧。
- 执行步骤。
- 赞成、保留、反对票数和共识率。

### 4.4 AI 降级

DeepSeek 失败时：

- 页面展示本地演示回复。
- 不声称回复来自在线模型。
- 所有人物投保留票。
- 共识率为 50%。
- 方案以防守、补充信息和小额验证为主。

## 6. 历史对话

- 发送第一条用户消息后自动保存。
- 标题取第一条用户问题前 22 个字符。
- 最多保存 20 个会话，按更新时间倒序。
- 保存内容：会话 ID、人物 ID、全部消息、最终方案、更新时间。
- 首页展示最近 6 个会话，并提供“继续最近对话”。
- 聊天侧栏展示最近 8 个会话。
- 恢复时同步恢复原人物组合、消息、投票和最终方案。
- 当前仅保存于浏览器，清除数据或更换设备后不会同步。

## 7. 人物社区

- 主导航“社区”取代旧“排行榜”。
- `/rankings` 自动跳转 `/community`。
- 社区顶部按 `uses` 排名，前三名大卡展示，其余列表展示。
- 社区支持人物、作者、流派和描述搜索。
- 官方和本地人物均可下载为 JSON。
- 上传只接受 JSON，导入后保存到当前浏览器，最多 30 个。
- 上传人物当前只进入社区，不自动加入首页委员会。

人物包格式：

```json
{
  "id": "unique-id",
  "name": "中文名称",
  "en": "English Name",
  "school": "投资流派",
  "quote": "人物核心语录",
  "risk": "稳健",
  "uses": 0,
  "author": "作者名称",
  "description": "人物分析框架说明"
}
```

`risk` 只允许：`稳健`、`均衡`、`激进`。

## 8. 市场面板

- 价格：DefiLlama Coins。
- TVL：DefiLlama Chains。
- 价格缓存约 60 秒，TVL 缓存约 300 秒。
- 当前数据源没有返回 24h 涨跌、市值或成交量时显示“—”，不填充假数据。
- 支持手动刷新。
- 每个资产可跳转到 AI 委员会并自动带入分析问题。
- 研究入口：OpenClue、DefiLlama、Dune、Nansen。

## 9. 个人中心与资金

### 9.1 模拟盘

- 初始余额：`$10,000`。
- 支持模拟充值和随时提现。
- 提现手续费：`amount * 0.0001`，即 0.01%。
- 提现时余额扣除“提现金额 + 手续费”。
- 流水最多保存 30 条。
- 资产趋势和链分布为界面演示，不代表真实持仓。

### 9.2 真实钱包

- MetaMask/EVM：读取 Ethereum ETH 和 BNB Chain BNB。
- Phantom/Solana：读取 SOL。
- 当前不读取 ERC-20、BEP-20、SPL、LP、借贷仓位或 NFT。
- 真实充值仅显示和复制用户钱包地址。
- 非托管资产由用户钱包直接控制。
- 收费金库尚未部署，真实提现不收取也不伪造 0.01% 手续费。

## 10. 钱包连接与执行

- EVM 使用 `window.ethereum` 和 `eth_requestAccounts`。
- Solana 使用 `window.solana.isPhantom` 和 `connect()`。
- 网站仅保存公开地址、钱包标签和钱包类型。
- 不读取或保存私钥、助记词。
- WalletConnect 尚未接入项目 ID 与 SDK。
- 方案执行页当前只进行本地模拟签名，延迟约 1.2 秒，不广播链上交易。

## 12. 当前不包含

- 真实 DEX、借贷或合约交易。
- 平台托管账户和收费金库。
- 自动投资 Keeper。
- WalletConnect、Privy 或社交登录。
- 服务端用户数据库和跨设备同步。
- 公共社区发布、审核、举报和评论。
- 生产级 IPFS/Filecoin 存储。
- KYT、地区限制和完整合规系统。

## 13. 生产前要求

- 智能合约和金库审计。
- 确定性交易 schema 与规则引擎。
- 协议白名单、滑点、价格影响、Gas、限额和授权检查。
- 钱包风险筛查、地区合规和用户声明。
- 速率限制、审计日志、监控与告警。
- 数据库、账户系统和加密同步。
